## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/mevaibhavpandey/Portfolio.git
```

### 2️⃣ Navigate to the project directory

```bash
cd Portfolio
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm start
```
